# Smart Financial Health Scorer

An end-to-end machine learning project that predicts company bankruptcy with **98.37% accuracy** using real financial data.

## Project Overview
This project analyzes 6,819 real company financial records to predict bankruptcy risk using XGBoost, SHAP explainability, and an interactive dashboard.

## Results
- **Accuracy:** 98.37%
- **Algorithm:** XGBoost Classifier
- **Dataset:** 6,819 companies, 96 financial features
- **Top Predictor:** Continuous interest rate (after tax)

## Tech Stack
- **Python** - Data analysis & ML
- **Pandas & SQL** - Data wrangling
- **XGBoost** - Machine learning model
- **SHAP** - Model explainability
- **Plotly** - Interactive dashboard
- **SQLite** - Database
